# Assets Directory

This directory contains assets for the GitBook documentation.

## Logo Configuration

GitBook logos are now configured to use assets from the root `/assets` directory:

- **Light Mode Logo**: `../../assets/gexd-text-logo-black.svg`
- **Dark Mode Logo**: `../../assets/gexd-text-logo-white.svg` 
- **Favicon**: `../../assets/gexd-favicon.svg`

## Available Assets in Root Directory

The main assets are located in `/assets/` and include:

- `gexd-favicon.svg` - Browser favicon
- `gexd-logo-transparent.svg` - Logo with transparent background
- `gexd-logo.svg` - Standard logo
- `gexd-simple.svg` - Simplified logo version
- `gexd-text-logo-black.svg` - Text logo for light backgrounds ✅ (Used in GitBook)
- `gexd-text-logo-white.svg` - Text logo for dark backgrounds ✅ (Used in GitBook) 
- `gexd-text-logo.svg` - Default text logo

## Notes

- All logos automatically adapt to light/dark theme changes
- The white logo glow effect has been removed for cleaner appearance
- GitBook configuration references assets using relative paths from the documentation root
